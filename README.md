Sou estudante de **Sistemas de Informação** e formado como **Técnico em Informática para Internet pelo Senac São Leopoldo**.

Atuo no desenvolvimento de soluções web, com foco crescente em **backend com Java e Spring Boot**, integração de APIs, bancos de dados e automação. Gosto de transformar necessidades reais em aplicações organizadas, funcionais e fáceis de manter.
